# A-Line-Translate-api

A Line Translate api 는 [한 줄 번역기](https://github.com/kyr9389/A-Line-Translate) 와 네이버 open api 통신을 위해 구축된 프로젝트입니다.

번역에 필요한 간단한 api를 제공합니다.

---
## 사용 방법

### translate

번역할 영어 문자열을 매개변수로 사용합니다. 반환 값은 한글 문자열입니다.

```
translate(string) 
```

## 기술 스택
- Node.js
